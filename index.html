¿Te cuesta llegar a las entregas?

Con este curso de 3 días, aprenderás técnicas y bases teóricas que te permitirán escribir código más rápido y de mejor calidad. Esto te permitirá convertirte en el programador top que siempre soñaste ser.

Bootstrapping

Todo lo necesario para que tu proyecto tenga dependencias claras, una marcada separación de capas y sea fácil de testar.

Autolayout y Size Classes

Con pantallas que van desde 4’ a 12.9’ con contenido flexible, cada vez es más importante escribir buen código para el layout.

Stateful View Controllers

Con esta técnica, será más fácil manejar la complejidad en un ViewController en escenarios donde la data debe ser obtenida desde la red.

Advanced Networking

Usaremos propiedades de Swift, como extensiones de protocolos, genéricos y enums para tener una capa de networking mas legible

Promises

En esta sección aprenderás una técnica para eliminar bugs producido por threading y poder realizar una serie de peticiones a la web antes de presentarle el resultado al usuario.

Core Data

Tests

Aprende a estructurar tu código para realizar tests que de verdad le den valor a tu proyecto. Cubriremos tanto Unit Tests, como Snaphosts y UI Tests.

Soft Skills
